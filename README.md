Twitterから「コロナ」という単語を含むツイートを取得し、
ネガポジ判定、トピックモデリングを自動で行うpythonプログラムです。

# Requirement
* Python 3.7~
* PyLDAvis 3.3.1
* snscrape 0.4.3.20220106
* sengiri 0.2.1
* gensim 4.2.0
* ipadic 1.0.0
* mecab-python3 1.0.5

**cd stulab3_project**

**python main.py options**

# option
* -h, --help: show help
* --reset: retry twitter scraping
